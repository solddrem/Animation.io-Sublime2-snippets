Animation.io-Sublime2-snippets
==============================

Sublime snippets for Animation.io Framework

# Usage

When editing an Animation.io-scene-file, enter:

* `actor [TAB]` for full actor definition.
* `videoactor [TAB]` for full videoactor definition.
